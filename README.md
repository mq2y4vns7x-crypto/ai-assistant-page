# ai-assistant-page # AI Assistant Webpage

This is a simple, responsive web wrapper for an AI Assistant powered by Jotform. 

## Features
* **Fullscreen Interface:** Uses a seamless iframe to provide a clean user experience.
* **Responsive Design:** Works on desktops, tablets, and mobile devices.
* **Permissions:** Configured to allow microphone and camera access for the AI agent.

## How to Use
1. Clone this repository.
2. Open `index.html` in any web browser.
3. (Optional) Replace the `src` URL in the `<iframe>` tag with your own Jotform Agent link.

## Deployment
This project is designed to be hosted easily via [GitHub Pages](https://pages.github.com/).
